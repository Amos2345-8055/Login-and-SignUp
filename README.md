# Login-and-SignUp
Introduction
This mini project focuses on developing a basic Login and Sign-Up system using the C programming language. The main objective of the project is to understand how user authentication works by storing and verifying user credentials such as usernames and passwords.
In this system, new users can create an account through the sign-up process by providing required details, which are then stored in a file. Existing users can log in by entering their credentials, which are validated against the stored data. The project demonstrates fundamental concepts of C programming, including file handling, structures, string manipulation, and conditional statements.
This project is useful for beginners to gain practical knowledge of how authentication systems work in real-world applications, while also improving problem-solving and programming skills.
Objective
To develop a basic Login and Sign-Up system using the C programming language.
To understand how user authentication works by verifying username and password.
To practice string handling, conditional statements, and input/output operations in C.
To perform a simple calculation (average of marks) after successful login.
Methodology
The program starts by asking the user to sign up.
User enters name, age, and password.
The system checks:
Age must be greater than 18
Password and confirm password must match
After successful registration, the user proceeds to login.
The entered login credentials are validated with registered data.
If login is successful:
User enters two subject marks
Program calculates and displays the average
If login fails, an error message is displayed.
